# Описание

Роль сервиса Gitea.

Создает на удаленном хосте пользователя git
Для созданного пользователя создает ssh ключ и добавляет его в authorized_keys на удаленном хосте.

Поднимает на удаленном хосте docker-compose с Gitea.
Пробрасывает web-интерфейс Gitea на 3000 порт, а ssh контейнера с Gitea на 2222 порт.

Настраивает NGINX удаленного хоста на проксирование {{ gitea_domain }}:80 на web-интерфейс Gitea.

Пробрасывает {{ server_data_dir }}/gitea в контейнер для хранения данных Gitea.

## Переменные

gitea_domain - доменное имя, на котором должен располагаться Gitea
server_data_dir - папка, в которой будет создана папка **gitea**, где Gitea будет хранить данные.
